zakuro's eslint config for vue.

## Install

Install config with required plugins.

```
npm install --save-dev @zakuro9715/eslint-config-vue
npx install-peerdeps --dev @zakuro9715/eslint-config-vue
```

## Configure

```.eslintrc
module.exports = {
  extends: ['@zakuro9715/vue']
}
```
