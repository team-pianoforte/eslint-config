zakuro's eslint config.

## Install

Install config with required plugins.

```
npm install --save-dev @zakuro9715/eslint-config eslint-plugin-promise eslint-plugin-import
```

## Configure

Add it to your .eslintrc

```.eslintrc
module.exports = {
  extends: ['@zakuro9715']
}
```
