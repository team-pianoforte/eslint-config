Pianofore style standard eslint config.

## Install

Install config with required plugins.

```
npm install --save-dev @pianoforte/eslint-config eslint-plugin-promise eslint-plugin-import
```

## Configure

Add it to your .eslintrc

```.eslintrc
module.exports = {
  extends: ['@pianoforte']
}
```
