zakuro's eslint config for nuxt.

## Install


```
npm install --save-dev @zakuro9715/eslint-config-nuxt
npx install-peerdeps --dev @zakuro9715/eslint-config-nuxt
```

## Configure

```.eslintrc
module.exports = {
  extends: ['@zakuro9715/nuxt']
}
```
